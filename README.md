# Grokking Algorithms

This is the code of Aditya Bhargava's book [Grokking Algorithms](https://www.manning.com/bhargava). All tasks are solved using Java.

Check out [Python Tutor](http://pythontutor.com/), a great website that guides you through Python code line by line.
